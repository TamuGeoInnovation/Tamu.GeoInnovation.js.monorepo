 How do I grant this application location access?

Are you seeing a location forbidden message when you try to submit a response? This can happen because either the location access was declined or because of system settings. Location is required to participate because we are interested in collecting geographic data on the submissions. Below are a few pointers to grant this application location access.

Trigger location access prompt: If you never saw the application request location access, this is a good first step. Press the button below to initiate geolocation which should prompt your device to ask you to allow this site access to location while the application is being used.

{{(locationRequested | async) ? 'Location Requested' : 'Request Location'}}

If the above resulted in an error, continue onto the following steps.

Enable System Location Services: Check that your device location services are enabled in system settings and enabled for Safari sites:

1. In your Settings app, tap on the Privacy link:

2. Tap on Location Services:

3. Tap on Safari Websites:

4. Ensure that location access is enabled While Using the App and enable Precise Location:

Enable Location in Safari Settings: Check that location is not disabled for Safari:

1. In your Settings app, tap on the Safari link:

2. Tap on the Location link:

3. Ensure that location access is not set to Deny. Set it to Ask for more convenient security:

With all those steps completed, your device should now be able to grant location access to the GIS Day Competitions App. Verify with the Request Location button above.

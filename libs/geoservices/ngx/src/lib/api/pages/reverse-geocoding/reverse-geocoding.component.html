Reverse geocoding is the process of converting a geographic coordinates into non-spatial locational data, most commonly postal address data. The services available on this site allow a user to process a single set of geographic coordinates as well as a database of records in batch.

Response

The following represents the response structure when requesting JSON format.

version
ApiVersion
 Describes the full API version that processed the transaction, reporting the major, minor, and patch version.

Number
 Time required to execute the transaction, reported in milliseconds.
transactionGuid
String
 Unique ID assigned to the response.

String
 Server name that processed the transaction.
clientHost
String
 Public hostname where the transaction was executed.

Enumerable<QueryStatusCode>
 The value for the response status code.

Array<ReverseGeocode>
 Result reverse geocode.

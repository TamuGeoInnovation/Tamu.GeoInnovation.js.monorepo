Aggiemap Beta

Thank you for checking out the Aggiemap Beta experience! The Aggiemap team is constantly looking to improve and add features to the platform to increase its usefulness, accuracy, and reliability. This branch is where we roll out the latest and greatest features and fixes that aren't quite production-ready. If you like to live dangerously, feel free to use this version of Aggiemap and get a taste for some of the upcoming features before they become available to everyone else.

Here is a brief summary on some of the notable WIP features:

3D Mode and Building Models

A feature long in the making, 3D mode has finally arrived! On the top left is a new perspective toggle widget that will switch between 2D and 3D perspective modes. When the 3D mode is enabled, so will a brand new layer displaying detailed building exterior shapes painstakingly authored to highlight the natural beauty of our campus. Together with 3D mode, it enables users to tilt and rotate the map view paving the way for a one of a kind experience and vast possibilities for future development efforts.

Bus Routes and Timetable Integration

On the sidebar, there is now a brand new tab with the bus icon (directions_bus) that contains a full list of all bus routes managed by Transportation Services. Clicking on a route will expand a timetable and display the bus route and live bus locations on the map, all without leaving Aggiemap.

Trip Planner Support for Parking Passes
The driving travel mode now allows users to configure their parking pass and the trip planner will try to generate directions to/from available parking lots based on time of day, permit selection, and other rule-based criteria.
Aggiemap is Open Source
There are no secrets here. If you are a developer and want to contribute towards improving the Aggiemap experience or simply want to see what makes Aggiemap tick, you can find the project on Github.

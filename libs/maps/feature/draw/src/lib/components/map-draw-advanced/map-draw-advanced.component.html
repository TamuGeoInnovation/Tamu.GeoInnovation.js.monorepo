Drawing tools help guide

open_with Transform tool: When a graphic is selected, this tool allows the shape to be re-positioned, rotated, and warped by the handles around the bounding box. This is the default tool. A shape can be de-selected by clicking on an empty area on the map or by pressing the ESC keyboard key.

format_shapes Reshape tool: Enables editing (move and delete) of individual shape vertices. Vertices can be moved by dragging them around and deleted by selecting and using the DELETE keyboard key.

room Point tool: Creates a simple point marker/graphic.

timeline Polyline tool: Creates a line made up of many points. Double click to complete line.

star_border Polygon tool: Creates a completely enclosed freeform polygon. Double click to complete shape.

check_box_outline_blank Rectangle tool: Creates a rectangle/square. Hold CTRL keyboard key while drawing/resizing to lock aspect ratio.

radio_button_unchecked Circle tool: Creates a circle/oval. Hold CTRL keyboard key while drawing/resizing to lock aspect ratio.

format_color_fill Color tool: Sets the color for the selected shape or next drawn shape.

undo Undo tool: When available, undos the last drawing operation.

redo Redo tool: When available, redos the last undone drawing operation.

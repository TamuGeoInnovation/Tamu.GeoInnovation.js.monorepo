Aggie Map Usage Instructions

Accessible Usage

Following university policy, Aggie Map is committed to making all features and information is accessible to all users. This application provides accommodations for screen readers and keyboard navigation. This page is provided for users requiring guided instruction through screen readers.

Visual Layout

Aggie Map is an interactive mapping application. The biggest focus is the map which covers the majority of the screen.

The map controls are found on the top left corner laid out from top to bottom. Each has a specific function. Their tab order and function are as follow:

  • Zoom In
  • Zoom Out
  • Location Tracker — Triggering it fetches the location of your device and adds a point on the map.
  • Orientation Compass — Triggering it resets map orientation if rotation is other than default.
  • Print — Triggering it opens a print dialog with exporting and printing options.

The right-hand side of the application houses a sidebar with two tabs. Each tab can be triggered to view the content described by the tab label.

The first tab labeled "Features" contains feature search and general map information. These include, in tab order from top to bottom:

  • Input search field — Begin typing here and receive auto-complete suggestions accessible with the up/arrow keyboard keys.
  • Layers — List of additional map layers which add or remove graphics on the map upon triggering them.
  • Legend — List of symbology icons and descriptions used in map to represent features and land use.

The second tab labeled "Directions" contains way-finding functionality: These include, in tab order from top to bottom:

  • Starting point input field — There are multiple ways to set a starting point, typing and selecting from dropdown is easiest for users depending on keyboard navigation.
  • Ending point input field — There are multiple ways to set a destination point, typing and selecting from dropdown is easiest for users depending on keyboard navigation.
  • Routing mode selection — This element consists of several horizontally laid out icons with each setting a different routing mode. These are, from left to right, walking and cycling.
  • Accessible friendly directions routing mode checkbox — Enables or disables routing through accessible paths. Defaults to disabled. Only available while routing in walking mode.

The selected feature popup also resides on the right-hand side but it is not part of the sidebar and does not have its own independent tab since popups can be dismissed completely. Selecting a feature is covered further in this text. The tab order is as follows:

  • Close — Trigerring will dismiss the popup re-focus on the map.
  • Feature name
  • Numerical feature code
  • Feature address
  • Feature permalink — URL unique to this building. Providing this URL in the browser address bar will load map with feature centered on it.
  • Copy URL — Convenient button, that when triggered, will copy the aforementioned permalink to your clipboard.
  • Directions To Here — Triggering will attempt to compute the best route between your current location gathered from your device and the selected feature.
  • Feature Image — If available, an image of the feature.

Keyboard Navigation

Keyboard controls have been added to this application, making it possible for users requiring on keyboard navigation to access all of the features provided. In addition, the tabbing order for Aggie Map has been optimized for easy and fast navigation.

Basic navigation is as follows:

TAB — Go to next element.

SHIFT + TAB — Go to previous element.

ENTER OR SPACE — Simulate a click on elements that have functionality associated with them.

UP OR DOWN arrow keys in input fields — If the input field generates suggestions as you type, these keys will focus on that list.

UP, RIGHT, DOWN, LEFT arrow keys on map — Pan the map in the direction of the arrow key (North, East, South, West respectively).

A OR D keys on map — Rotate the map. A rotates clockwise. D rotates counter-clockwise.

Selecting a map feature can be done by setting focus on the map and begin panning the map with the keyboard arrow keys. A viewfinder will become visible on the center of the screen with a white box below it describing the features within the viewfinder. As you pan, the list will update. Each feature on the list will carry a number besides it corresponding to a keyboard number. Pressing the respective number will select the feature, highlight it on the map, and open the popup.
